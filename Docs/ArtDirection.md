# Sol City art direction

The shipped source textures were generated with the built-in OpenAI image generation tool and are intended for box/HISM city massing. The visual target is a peaceful, vivid Japanese-animation background rather than photorealism.

## Generated asset prompts

- `T_AnimeGrass.png`: seamless square, top-down spring grass and tiny clover, crisp hand-painted anime background texture, diffuse daylight, no objects/paths/text/border.
- `T_AnimeAsphalt.png`: seamless square, top-down cool blue-gray asphalt with subtle aggregate, low visual noise, no markings/cracks/curb/text/border.
- `T_AnimeWater.png`: seamless square, top-down turquoise river with horizontal ribbon highlights for UV panning, no banks/objects/text/border.
- `T_AnimeFacade.png`: seamless orthographic warm-ivory apartment facade, repeating windows and pastel balcony accents, no perspective/signage/people/text/border.

The full production constraints used for every image were: square edge-to-edge surface, no horizon or focal object, peaceful bright daylight, no watermark, and visually repeatable opposite edges.
